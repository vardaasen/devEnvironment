## Oversikt

### Avvik (AVV)

Avvik dokumenterer uventede funn under utvikling — ting som ikke fungerte som forventet, og hvordan de ble løst. Nyttig for fremtidig feilsøking og som læringsressurs.

### Beslutninger (ADR)

Architecture Decision Records dokumenterer viktige tekniske valg med kontekst, alternativer vurdert, og begrunnelse. Noen er vedtatt, andre er under utredning.

| ADR | Tittel | Status |
|---|---|---|
| ADR-001–004 | Grunnleggende arkitektur (XDG, lagdeling, testing, line endings) | Vedtatt |
| ADR-005 | Admin-kjøring og sikkerhetsvurderinger | Vedtatt |
| ADR-006 | Identitets- og autentiseringsarkitektur | Under utredning |
| ADR-007 | Secrets and Identity Management Strategy | Under vurdering |
| ADR-008 | Container- og virtualiseringsstrategi | Under utredning |
| ADR-009 | Oppdateringsnotifikasjoner ved terminaloppstart | Utsatt |
| ADR-010 | Runtime-valg — Deno over Node.js, native Claude Code | Vedtatt |
